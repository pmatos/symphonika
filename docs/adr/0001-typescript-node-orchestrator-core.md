# TypeScript and Node for the orchestrator core

Symphonika will use TypeScript on Node.js for the fresh orchestrator implementation. Elixir/OTP was a strong fit for long-running supervision and Rust was a strong fit for hardened single-binary operation, but TypeScript gives the best balance for agentic coding: strict static checking, fast edit-test loops, excellent JSON/protocol ergonomics, mature GitHub tooling, subprocess streaming, and straightforward debugging by coding agents.
